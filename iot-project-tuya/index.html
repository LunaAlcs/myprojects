IoT Project - Monitoramento com Tuya API

Solução em nuvem para monitoramento ambiental (temperatura, umidade e bateria) com API Tuya, banco InfluxDB Cloud e dashboards em Grafana. O projeto usa integração com o aplicativo Smart Life e visualização de dados em tempo real via Grafana Cloud.

  • 📡 API REST Tuya com autenticação HMAC SHA256
  • ☁️ Banco InfluxDB Cloud para dados em série temporal
  • 📊 Dashboards interativos no Grafana Cloud
  • 🧠 Código Python para coleta, envio e integração
  • 🔗 Aplicativo Smart Life como ponte com o sensor
